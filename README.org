* Brightshift

Easily manage your screen brightness using DDC/CI.

A simple brightness app which uses ddc

Usage: brightness <COMMAND>

Commands:
  get-device
  get-brightness, -g
  set-brightness, -s
  help                Print this message or the help of the given subcommand(s)

Options:
  -h, --help     Print help
  -V, --version  Print version
